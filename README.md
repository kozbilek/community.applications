# community.applications - Modified for low RAM devices

## Overview

This is a fork of the original community.applications project by Squidly271 with modifications to bypass the minimum RAM requirement. The changes have been made to accommodate devices with limited RAM capacity, such as my Intel SS4200-E NAS with 2G of RAM.

## Important Note

While the application currently works smoothly with the available RAM on my device, it may not be stable on devices with different RAM configurations. Users are advised to try this modified version at their own risk.
